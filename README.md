# 📊 Emotional Analytics Dashboard (Power BI)

## 📌 Project Overview
An interactive Power BI dashboard built to analyze 
user emotions over time using a dataset of 1,200 records.

## 🛠️ Tools Used
- Power BI
- MS Excel (Data Cleaning)

## 📈 What I Did
- Cleaned and structured raw dataset of 1,200 records
- Identified patterns based on time, activities & mood
- Created bar, line, pie and KPI chart visualizations
- Delivered actionable insights on user behavior

## 📊 Key Results
| Metric | Value |
|--------|-------|
| Total Records | 1,200 |
| Positive Emotions | 620 |
| Neutral Emotions | 380 |
| Negative Emotions | 200 |
